# Education Signature Manager v1.6

법정 의무교육 전자서명 관리 프로토타입입니다.

## 실행 방법

압축을 해제한 뒤 `index.html`을 Chrome 또는 Edge에서 실행합니다.

## 관리자 로그인

- ID: `admin`
- PW: `1234`

## 주요 기능

- 직원 DB 엑셀 업로드 및 직원 직접 추가
- 직원 로그인: 이름 + 사번 + 휴대폰 번호 전체
- 교육 관리 메뉴 통합
- 법정교육 구분/연도/주기/세부 주기 기반 교육명 자동 생성
- 회차별 교육일자, 장소, 시작/종료시간, 상태 관리
- 회차별 대상자 별도 설정
- 1반, 2반, 주간 정취 선택 및 개별 체크로 대상자 설정
- 회차별 서명 현황 확인
- 전체 통합 서명 현황 확인
- Excel 서명부 내보내기
- PDF/인쇄용 서명부 출력
- 미서명자 CSV 다운로드
- 등록된 교육 기본정보 수정
- 회차별 수정/대상자 재설정
- 신규 교육 등록 버튼 제공

## v1.6 변경 사항

- 교육 대상자를 교육 단위가 아니라 회차 단위로 저장하도록 변경했습니다.
- 1차/2차/보충교육별 대상자와 서명 현황을 분리했습니다.
- 서명 현황/출력에서 회차별 보기와 전체 통합 보기를 분리했습니다.
- 회차 목록에 대상자 수와 서명 수를 표시합니다.
- 교육 수정 후 신규 등록으로 돌아갈 수 있도록 `신규 교육 등록` 버튼을 추가했습니다.
- 회차명 중복 표시 문제를 개선했습니다.

## 참고

현재 버전은 브라우저 localStorage 기반 프로토타입입니다. 실제 운영 시에는 Supabase 또는 사내 DB/서버 연동이 필요합니다.
